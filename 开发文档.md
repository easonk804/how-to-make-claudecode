# 开发文档

本项目采用 10 章认知递进式教学路线，核心面向对 Claude / Agent 内部机制不熟悉的小白和初学者。

## 教学原则

每章都必须包含以下五部分：
1. 问题引入（为什么加入这个机制）
2. 最小实现（几十行到一两百行即可跑）
3. 逐步完善（v1 -> v2 -> v3）
4. 可视化（ASCII 图）
5. 练习题（基础 / 进阶 / 挑战）

## 章节地图

1. 感知-思考-行动循环
2. 工具系统
3. 短期记忆
4. Chain-of-Thought / ReAct
5. 自我纠错
6. 外部记忆（RAG）
7. 上下文压缩
8. 并发与后台任务
9. 多 Agent 协作
10. 安全边界与沙箱
